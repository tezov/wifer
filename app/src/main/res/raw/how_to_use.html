How to use ?

Prerequisite

Both devices must be connected to the same network Wifi. The Wifi can be:

  • Local network Wifi
  • One device is connected to the other device shared wifi (Tether / Hotspot)

Send file

  • click Send button
  • select file to send
  • click Start button
  • when the transfer is done, you can select another file to send

Receive a file

  • click Receive button
  • click Start button
  • Wait for transfer completion

Notes

-Both Sender and Receiver must use the same Port.

-If Port is different, the Receiver can't find the Sender and any transfer will fail.

-Received files will be in your android Media folder(Pictures, Music, Movies or Documents)

-Port number must be between 49152 and 65535
