Comment ça marche ?

Prérequis

Les deux appareils doivent être connectés sur le même Wifi. Le Wifi peut être:

  • Wifi en réseau local
  • L'un des appareils est connecté au wifi partagé du second appareil (Tether / Hotspot)

Émetteur

  • cliquez sur le button Envoyer
  • selectionnez le ficher à envoyer
  • cliquez sur le button Start
  • Quand le transfert est fini, vous pouvez sélectionner un autre fichier à envoyer

Récepteur

  • cliquez sur le button Recevoir
  • cliquez sur le button Start
  • attendez la fin du transfert

Notes

-L'Émetteur et le Récepteur doivent utiliser le même Port.

-Si le Port est différent, le Récepteur ne pourra pas trouver l'Émetteur et le transfert ne pourra pas se faire

-Les fichiers réçus se trouveront dans l'un des dossiers android media(Pictures, Musics, Movies ou Documents)

-Le numéro de port doit être compris entre 49152 and 65535
